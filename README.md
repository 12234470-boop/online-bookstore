# Online Bookstore - React Application

A modern online bookstore with dark theme and search functionality.

## Live Demo
https://12234470-boop.github.io/online-bookstore

## Features
- 5 responsive pages
- Book search
- Dark modern design
- Shopping cart

## Setup
1. npm install
2. npm start
3. Open http://localhost:3000
